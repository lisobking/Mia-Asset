# Design System: Anti-Gravity Trading Bot

이 문서는 프리미엄 다크/글래스모피즘 기반의 UI/UX 디자인 일관성을 유지하기 위한 시스템 가이드라인입니다.

## 1. Colors (색상 팔레트)
- **Background (Dark)**: `#0a0b10`
- **Glass Panel**: `rgba(22, 25, 37, 0.65)` (Backdrop blur: 12px)
- **Accent (Neon Cyan)**: `#00f0ff`
- **Text Main**: `#f0f0f5`
- **Text Muted**: `#8b92a5`
- **Success (Green)**: `#00ff88`
- **Danger (Red)**: `#ff3366`
- **Warning (Yellow)**: `#ffcc00`

## 2. Typography (타이포그래피)
- **Main Font**: `Inter, sans-serif` (본문, 수치, 일반 텍스트)
- **Heading Font**: `Outfit, sans-serif` (로고, 제목, 강조 가격 수치)

## 3. Components (주요 컴포넌트 규격)
- **Glass Panel Radius**: `16px`
- **Button Radius**: `8px`
- **Badge Radius**: `20px`

## 4. Effects (효과)
- **Glow Effect (Neon)**: `box-shadow: 0 0 15px rgba(0, 240, 255, 0.3)`
